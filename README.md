##😊 Project

프로 팀플러들의 스토리브 리그, 팀모입니다.

팀모는 언택트 시대의 대학생들을 위한 팀 빌딩 + 팀 협업 서비스로, 팀으로 진행하는 활동에서 팀원을 찾기 어려워 고민하는 대학생들을 위해 만들어졌습니다. 팀모의 팀원 모집/찾기를 통해 나와 딱 맞는 팀원을 찾아보세요! 그리고 협업을 시작하세요!

<br>

## 🤔 Function

- 팀모 (팀원 모집)
- 팀글 (팀원 찾기)
- 마이페이지
